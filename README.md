# CCM Desktop Companion

This repository contains the standalone Windows desktop companion for Crafting Cooldown Manager.

Repository layout:
- `CCM.DesktopCompanion/` - WinForms tray application source
- `Installer/` - WiX MSI project
- `Builds/` - built MSI output intended for sharing
- `tools/` - local build scripts
- `artifacts/publish/` - local publish output generated by the MSI build script

Build the MSI:
```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\tools\Build-DesktopCompanionMsi.ps1
```

Launch the app locally for development:
```cmd
Launch-CCM-DesktopCompanion.cmd
```
